**Sword Demon** — це сучасний та динамічний лендинг для вигаданої гри у жанрі екшн/фентезі. Проєкт демонструє майстерність роботи з версткою, адаптивним дизайном та складними CSS-анімаціями.

---

## 🚀 Особливості (Features)

* **Atmospheric Design:** Темна кольорова гама, що ідеально підходить для ігрової тематики.
* **Fully Responsive:** Сайт коректно відображається на десктопах, планшетах та мобільних пристроях.
* **Smooth Animations:** Використання CSS-ефектів для занурення користувача.
* **Interactive Elements:** Стилізовані кнопки, ховер-ефекти та структуровані секції з інформацією про гру.
* **Clean Code:** Семантична верстка та організована структура файлів.

---

## 🛠️ Технологічний стек

* **HTML5** — семантична розмітка контенту.
* **CSS3** — стилізація, Flexbox/Grid для макету, Keyframes для анімацій.
* **Google Fonts** — для кастомної типографіки.
